## Installation (Frontend Projects)

General steps to run a frontend project from this account:

1. Clone the repository
```bash
git clone https://github.com/Mirzohiddev006/<repo-name>.git
cd <repo-name>
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run development server
```bash
npm start
# or (Next.js)
npm run dev
```

4. Build for production
```bash
npm run build
npm run start
```

## Recommended Frontend Practices

- Use semantic HTML and responsive CSS (mobile-first approach).
- Add accessibility checks (axe, Lighthouse) during development.
- Enforce code quality with ESLint + Prettier and type-safety with TypeScript.
- Add unit and integration tests for critical UI components (Jest, React Testing Library).
- Deploy static/SSR sites to Vercel, Netlify, or similar services.
- Use GitHub Actions for CI (lint, test, build) and add a status badge.
